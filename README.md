# Creator Connect

A modern full-stack creator marketplace platform where shop owners can connect and collaborate with content creators.

## 🚀 Features

- Modern responsive UI
- Creator & Shop Owner roles
- Secure authentication
- Login & Signup system
- Google authentication UI
- Password visibility toggle
- Redux state management
- Protected routes
- Animated UI using Framer Motion
- Form validation with React Hook Form

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Redux Toolkit
- React Router DOM
- Tailwind CSS
- Framer Motion
- React Hook Form
- Lucide React

### Backend
- Node.js
- Express.js
- MongoDB
- REST APIs

### Authentication
- Appwrite Authentication

## ⚙️ Installation

### 1 Clone Repository

```bash
git clone https://github.com/nooralam4919/creator-connect-app.git
```

### 2 Move into Project

```bash
cd creator-connect-app
```

### 3 Install Dependencies

```bash
npm install
```

### 4 Run Development Server

```bash
npm run dev
```

---

## 🔐 Environment Variables

Create `.env` file:

```env
VITE_APPWRITE_URL=your_appwrite_url
VITE_APPWRITE_PROJECT_ID=your_project_id
```
